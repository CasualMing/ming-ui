# 配置: loading-options

## loading

* 说明: 是否展示加载状态
* 类型: Boolean
* 可选值: false / true
* 默认值:  false

## text

* 说明: 加载状态文案
* 类型: String
* 可选值: 无
* 默认值: '拼命加载中'

## spinner

* 说明: 加载状态图标类名
* 类型: String
* 可选值: 无
* 默认值: 'el-icon-loading'

## background

* 说明: 加载状态背景色值
* 类型: String
* 可选值: 无
* 默认值: 'rgba(255, 255, 255, 0.5)'
